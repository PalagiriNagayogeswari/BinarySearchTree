# Binary Search Tree in Java

This repository contains Java implementations of **Binary Search Tree (BST)** operations. It is designed to help understand the structure, properties, and basic operations of Binary Search Trees using Java. The repository is useful for learning Data Structures & Algorithms (DSA) and preparing for coding interviews and technical assessments.

---

## Topics Covered

- Binary Search Tree
- Node Creation
- Insertion
- Searching
- Deletion
- Tree Traversals
  - Inorder Traversal


---

## How to Run

### Clone the repository

```bash
git clone https://github.com/PalagiriNagayogeswari/BinarySearchTree.git
```

### Navigate to the project directory

```bash
cd BinarySearchTree
```

### Compile a Java file

```bash
javac FileName.java
```

### Run the program

```bash
java FileName
```

Replace `FileName` with the name of the Java file you want to execute.

## Time Complexity

| Operation | Average Case | Worst Case |
|-----------|--------------|------------|
| Search | O(log n) | O(n) |
| Insert | O(log n) | O(n) |
| Delete | O(log n) | O(n) |
| Traversal | O(n) | O(n) |

> The average-case complexity assumes the Binary Search Tree is reasonably balanced.
